# R Strict Quality Config

Strict mode configuration for R projects with lintr and styler.

## Mode: Strict

- Coverage: 80% minimum
- Complexity: Standard limits
- lintr with recommended linters

## styler Config (`.Rprofile` or `styler.R`)

```r
# styler configuration for strict mode
options(
  styler.addins_style_transformer = "styler::tidyverse_style(
    indent_by = 2,
    strict = TRUE
  )"
)

# Project-specific styler configuration
styler_strict <- function() {
  styler::tidyverse_style(
    scope = "tokens",
    indent_by = 2L,
    strict = TRUE,
    start_comments_with_one_space = TRUE,
    reindention = styler::specify_reindention(
      regex_pattern = NULL,
      indention = 0L,
      comments_only = TRUE
    ),
    math_token_spacing = styler::specify_math_token_spacing()
  )
}
```

## Project Scripts (Makefile)

```makefile
.PHONY: lint style test check coverage install

lint:
	Rscript -e "lintr::lint_package()"

style:
	Rscript -e "styler::style_pkg(strict = TRUE)"

style-check:
	Rscript -e "if (any(styler::style_pkg(dry = 'on')[[1]][['changed']])) stop('Code needs styling')"

test:
	Rscript -e "testthat::test_local()"

check:
	R CMD check --as-cran .

coverage:
	Rscript -e "covr::package_coverage(quiet = FALSE)"

coverage-report:
	Rscript -e "cov <- covr::package_coverage(); covr::report(cov)"

coverage-check:
	Rscript -e "cov <- covr::package_coverage(); pct <- covr::percent_coverage(cov); if (pct < 80) stop(sprintf('Coverage %.1f%% below 80%%', pct))"

install:
	R CMD INSTALL --no-multiarch --with-keep.source .

quality: lint style-check coverage-check
	@echo "All quality checks passed"
```

## GitHub Actions Workflow (`.github/workflows/R-CMD-check.yaml`)

```yaml
name: R-CMD-check

on:
  push:
    branches: [main, master]
  pull_request:
    branches: [main, master]

jobs:
  R-CMD-check:
    runs-on: ${{ matrix.os }}
    strategy:
      fail-fast: false
      matrix:
        os: [ubuntu-latest, macos-latest, windows-latest]
        r-version: ['4.3', '4.4']

    steps:
      - uses: actions/checkout@v4

      - uses: r-lib/actions/setup-r@v2
        with:
          r-version: ${{ matrix.r-version }}
          use-public-rspm: true

      - uses: r-lib/actions/setup-r-dependencies@v2
        with:
          extra-packages: any::rcmdcheck, any::lintr, any::covr
          needs: check

      - name: Lint
        run: lintr::lint_package()
        shell: Rscript {0}

      - name: Check
        uses: r-lib/actions/check-r-package@v2

      - name: Coverage
        if: matrix.os == 'ubuntu-latest' && matrix.r-version == '4.4'
        run: |
          cov <- covr::package_coverage()
          covr::codecov(coverage = cov)
        shell: Rscript {0}
        env:
          CODECOV_TOKEN: ${{ secrets.CODECOV_TOKEN }}
```
